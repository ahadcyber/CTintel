üéØ Overview

The Cyber Threat Intelligence (CTI) Dashboard is a comprehensive, production-ready platform designed for collecting, analyzing, and visualizing threat intelligence data from multiple free sources. This tool empowers security researchers, analysts, and organizations to stay ahead of emerging threats.

üöÄ How It Works - Step by Step

‚ö° Real-Time Data Pipeline

  1. Step 1 - Data Collection: Python scripts automatically fetch threat intelligence from 5 public sources (ThreatFox, PhishTank, Spamhaus, OTX, AbuseIPDB) every 30 minutes
  2. Step 2 - Processing: Each IOC (malicious IP, URL, domain, or file hash) is validated, deduplicated, and timestamped
  3. Step 3 - Storage: Data is stored in MongoDB Atlas cloud database with automatic indexing for fast queries
  4. Step 4 - Analysis: Flask backend calculates statistics, trends, and threat levels in real-time
  5. Step 5 - Visualization: Interactive dashboard displays IOCs with Chart.js graphs, search functionality, and export options

üîç Threat Lookup Process

1. Enter any IP address, domain, URL, or file hash
2. System checks local database (2000+ IOCs) for matches
3. Simultaneously queries VirusTotal API with your key
4. Displays threat level (Clean/Low/Medium/High/Critical) based on detection engines
5. Shows detailed analysis: malicious detections, suspicious flags, harmless ratings
